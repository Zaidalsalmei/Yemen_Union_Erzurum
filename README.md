### 🌟 رؤية المشروع (Project Vision)
نظام **YEME System** هو منصة إدارية متكاملة (ERP) صممت خصيصاً لرقمنة وإدارة شؤون الطلاب اليمنيين في مدينة أرضروم التركية. يمثل النظام نقلة نوعية من العمليات الورقية التقليدية إلى بيئة رقمية متكاملة تضمن الشفافية الإدارية والمالية.

### 🎯 أهداف النظام الأساسية (Mission & Goals)
* 🗄️ **حوكمة البيانات:** إدارة 86 جدولاً من البيانات الطلابية والأكاديمية والمالية بشكل مترابط.
* 🔐 **الأمان والخصوصية:** تطبيق معايير أمنية صارمة لحماية بيانات الطلاب وإدارة الصلاحيات (RBAC).
* 💰 **الكفاءة المالية:** تتبع دقيق للاشتراكات السنوية ونصف السنوية مع توليد تقارير مالية فورية.
* 📱 **التواصل الرقمي:** توفير نظام محادثة داخلي متطور وتكامل مع Telegram و Gmail لضمان سرعة وصول الإشعارات.

### 🚀 الميزات المتقدمة (Core Features)
- **🛡️ Security Core:** نظام تشفير متقدم وإدارة جلسات (JWT) مع سجل مراقبة شامل (Audit Logs).
- **💬 Integrated Chat:** نظام مراسلة فورية يدعم المحادثات الفردية والجماعية داخل المنصة.
- **🏦 Finance Engine:** محرك مالي مخصص لمعالجة الاشتراكات، المصروفات، والتقارير المالية للاتحاد.
- **📧 Auto Notifier:** تنبيهات تلقائية عبر البريد الإلكتروني وبوت التليجرام لكل الأحداث المهمة.
