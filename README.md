# 🇾🇪 نظام اتحاد الطلاب اليمنيين في أرضروم (YEME System)

نظام إدارة متكامل (ERP) مصمم لتنظيم شؤون الطلاب اليمنيين في مدينة أرضروم، يتميز بمعايير أمان عالية وإدارة ذكية للصلاحيات.

## 🛡️ الميزات الأمنية (Security Core)
* **نظام الأكواد المخفية (Hidden Role OTP):** يتم منح الصلاحيات (رئيس/عضو) بناءً على كود التحقق الإداري.
* **تكامل Telegram Bot:** تصل التنبيهات والأكواد مباشرة للمسؤول على التلجرام.
* **التحقق عبر البريد الإلكتروني:** استخدام Gmail SMTP لاستعادة كلمة المرور بأمان.

## 🚀 التقنيات (Tech Stack)
* **Frontend:** React.js + Tailwind CSS.
* **Backend:** PHP.
* **Integrations:** Telegram Bot API & Gmail SMTP.
