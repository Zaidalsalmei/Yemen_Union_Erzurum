### 🔍 **نظام البحث والفلترة:**

### 📋 **الجدول الاحترافي:**

| العمود | المحتوى |
|--------|---------|
| العضو | الاسم + الصورة الرمزية + الهاتف |
| نوع الاشتراك | الاسم + المدة |
| تاريخ البداية | تاريخ بداية الاشتراك |
| تاريخ الانتهاء | التاريخ + الأيام المتبقية |
| المبلغ | المبلغ بالعملة |
| الحالة | Badge ملون (ساري/منتهي/ينتهي قريباً) |
| الإجراءات | عرض + تجديد |

### ⚠️ **التنبيهات الذكية:**
- 🟢 **ساري** - اشتراك نشط (أكثر من 30 يوم)
- 🟡 **ينتهي قريباً** - خلال 30 يوم أو أقل
- 🔴 **منتهي** - انتهى الاشتراك

### 🔄 **Pagination:**
- عرض 10 اشتراكات لكل صفحة
- أزرار السابق/التالي
- أرقام الصفحات
- عرض النطاق (من X إلى Y من Z)

const isActive = new Date(membership.end_date) > new Date();

### 2. **حساب الإحصائيات:**
```typescript
const stats = data?.data.reduce((acc, membership) => {
  const isActive = new Date(membership.end_date) > new Date();
  if (isActive) {
    acc.active++;
    acc.revenue += membership.amount;
  } else {
    acc.expired++;
  }
  acc.total++;
  return acc;
}, { total: 0, active: 0, expired: 0, revenue: 0 });
```
