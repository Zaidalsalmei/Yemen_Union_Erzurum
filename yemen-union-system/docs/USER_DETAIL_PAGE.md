## 🔗 الارتباطات:

### روابط داخلية:
- `/users` - العودة للقائمة
- `/users/:id/edit` - تعديل البيانات
- `/memberships/create?user_id=:id` - إضافة اشتراك

### API Endpoints:
- `GET /users/:id` - جلب بيانات العضو
- `DELETE /users/:id` - حذف العضو
