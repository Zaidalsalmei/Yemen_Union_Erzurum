## 🎯 مميزات صفحة تفاصيل العضو:

### Layout:
```
┌─────────────────────────────────────┐
│  Header (العنوان + الأزرار)         │
├──────────┬──────────────────────────┤
│          │                          │
│ Profile  │  المعلومات الشخصية      │
│  Card    │                          │
│          ├──────────────────────────┤
│  (صورة   │  المعلومات الدراسية      │
│  معلومات │                          │
│  أزرار)  ├──────────────────────────┤
│          │  معلومات الاشتراك        │
│          ├──────────────────────────┤
│          │  الأدوار والصلاحيات      │
└──────────┴──────────────────────────┘
```

### الأقسام:
1. **بطاقة الملف الشخصي** (يسار)
   - صورة رمزية كبيرة

   - معلومات اتصال سريعة
   - أزرار الإجراءات

2. **المعلومات التفصيلية** (يمين)
   - المعلومات الشخصية (6 حقول)
   - المعلومات الدراسية (6 حقول)
   - معلومات الاشتراك (مع تنبيهات)
   - الأدوار والصلاحيات

### الإجراءات:
- ✏️ تعديل البيانات
- 🗑️ حذف العضو (مع تأكيد)
- ✅ الموافقة على العضوية (pending فقط)
